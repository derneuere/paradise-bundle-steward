# Prop Types (Burnout Paradise Remastered)

**Source:** [Burnout wiki — Prop Physics / Prop Types](https://burnout.wiki/wiki/Prop_Physics/Prop_Types), captured 2026-06-08.

These are the prop type definitions referenced by [`PropInstanceData`](prop-instance-data-spec.md)
(`0x10011`). A `PropInstanceData.muTypeIdAndFlags` packs a **type ID in the lower
26 bits**; that type ID is an **index into this table** (the `ID/Index` column).
`muAlternativeType` is likewise an index into this table (or `0xFFFF` for "none").

The machine-readable form of this table is baked into
[`src/lib/core/propTypes.ts`](../src/lib/core/propTypes.ts) and surfaced as the
prop-type dropdown in the PropInstanceData editor. **Keep the two in lockstep** —
`propTypes.ts` is generated from this table (see the test that asserts the count).

The columns are: **Index** (the type ID), **Resource ID** (the prop resource's
bundle id, hex), **GameDB ID**, and **Name**.
